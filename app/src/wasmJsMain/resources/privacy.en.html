3. Server Operation, Technical Connection Data and IP Addresses

I do not conduct independent IP tracking and do not analyze IP addresses for marketing purposes. At the application level, the service is configured so that IP addresses and technical connection data are not deliberately stored permanently.

Independently of this, the hosting provider (Hetzner) inevitably processes technical connection data, in particular IP addresses, as part of the technical operation and security of the infrastructure, to the extent necessary for provision, operation, error analysis and protection against attacks. Which specific technical log data Hetzner stores in individual cases and for how long depends on the systems used and the specifications of the hosting provider. This data is generally not provided to me.

Categories of data processed: Technical connection data (IP address, timestamp, HTTP request data, browser information).

Legal basis: Art. 6(1)(f) GDPR (legitimate interest in secure and reliable server operation and protection against abuse).

Obligation to provide: The processing of technical connection data is mandatory for accessing the website. Without this data processing, the service cannot be provided.

Third country transfer: No transfer of personal data to third countries outside the EU or EEA takes place as part of hosting.

Further information on data processing by Hetzner can be found in Hetzner's privacy policy: https://www.hetzner.com/legal/privacy-policy/

4. Steam Login (Optional) and Third-Party Services

The ONI Seed Browser can generally be used without logging in. Optionally, you can log in via Steam Login.

An external authentication service from Steam (Valve Corporation, USA) is used for login. Communication takes place between your device and Steam. I only receive your Steam ID. No further Steam profile information (e.g., username, avatar, or friends list) is retrieved from Steam.

No separate user account is created with this service. Authentication is performed exclusively via Steam, and all data is directly linked to your Steam ID.

Categories of data processed: Online identifiers (Steam ID, cryptographic hash of Steam ID).

Recipients: As part of the login process, data is transmitted to Steam or Valve Corporation (USA). I subsequently process the Steam ID received from Steam to perform the login and for the functions described below.

Third country transfer: As part of Steam authentication, data is transferred to Valve Corporation, USA. Valve has committed to complying with EU standard contractual clauses. Further information can be found in Steam's privacy policy: https://store.steampowered.com/privacy_agreement/

For technical authentication, a JWT (JSON Web Token) is generated, which contains the Steam ID and a cryptographic hash of this ID.

To remain logged in, this JWT can be stored locally in the browser (see section "Storage of Settings and Login Status (localStorage)").

Purpose: Authentication, abuse prevention, and assignment of contributions and requests.

Legal basis: Art. 6(1)(a) GDPR (consent) and Art. 6(1)(f) GDPR (abuse prevention).

Withdrawal of consent: You can withdraw your consent at any time by deleting the website data (localStorage) in your browser. Withdrawal does not affect the lawfulness of processing carried out before the withdrawal.

Obligation to provide: Login via Steam is voluntary. Without logging in, you can use the service in read-only mode, but you cannot upload Seeds/Maps or make requests.

5. Seeds/Maps, Mod Uploads, Requests and Leaderboard

Users can submit Seeds/Maps to the service API via a mod for the game Oxygen Not Included. A valid Steam auth token (JWT) is required for this.

Additionally, logged-in users can request Seeds/Maps to be added to the database (requests), and optionally set a name for a leaderboard.

The following information is processed in the database:

  • The submitted seed or map data structure (contribution data)
  • An assignment to the Steam ID of the contributing user (for abuse prevention and traceability)
  • If voluntarily provided: a user-defined name for the leaderboard

Categories of data processed: Online identifiers (Steam ID, cryptographic hash), game data (Seeds/Maps), optional user input (leaderboard name).

Leaderboard: The leaderboard shows which user has contributed how many Seeds/Maps. Public assignment to a person is only possible if a user voluntarily provides a name for the leaderboard. You can remove yourself from the leaderboard at any time by entering an empty name.

Public delivery: Seeds/Maps are delivered publicly. Not the Steam ID itself, but exclusively a hash of the Steam ID is used, so that no direct assignment to a person is possible.

Storage duration:

  • For requested Seeds/Maps (requests), the Steam ID is deleted from the database once the requested seed or map has been successfully generated or added.
  • For Seeds/Maps submitted via the mod, the Steam ID remains stored together with the respective contribution for the duration of its existence. No automatic deletion occurs. Upon request of the contributor, a specific contribution (Seed/Map) can be deleted. The deletion removes the assignment of this contribution to the Steam ID.
  • Since no separate user account is created, there is no central "account deletion". Deletion occurs on a contribution-by-contribution basis upon request (see section "Your Rights").

Legal basis: Art. 6(1)(a) GDPR (consent for authenticated uploads) and Art. 6(1)(f) GDPR (abuse prevention, system integrity, traceability of contributions).

6. Storage of Settings and Login Status (localStorage)

The ONI Seed Browser stores certain data locally in the browser (localStorage). This storage occurs exclusively on your device.

a) Search filters and markers
Users can save search filters and mark coordinates. This information is stored exclusively locally and is not transmitted to the server.

b) Login status (JWT)
When you log in via Steam, the generated authentication token (JWT) is stored locally in the browser so that you remain logged in and can use the corresponding functions. The JWT does not have an automatic time limit. No dedicated logout function is implemented. To end your login session, delete the website data in your browser (delete localStorage).

The JWT is transmitted to my server during authenticated API requests to assign your Steam ID and provide the desired functions.

Storage duration:
Data stored in localStorage remains there until you delete it yourself in the browser.

Legal basis (device storage):
§ 25(2) No. 2 TTDSG (technically necessary storage/access to provide a function requested by the user).

Legal basis (data processing):
Art. 6(1)(a) GDPR (consent) and Art. 6(1)(f) GDPR (abuse prevention).

7. Encryption

All data transmission is encrypted via HTTPS (TLS).

8. Your Rights

Under the GDPR, you have the following rights:

  • Access (Art. 15 GDPR)
  • Rectification (Art. 16 GDPR)
  • Erasure (Art. 17 GDPR)
  • Restriction of processing (Art. 18 GDPR)
  • Data portability (Art. 20 GDPR)
  • Objection (Art. 21 GDPR)
  • Withdrawal of consent (Art. 7(3) GDPR)
  • Complaint to a supervisory authority (Art. 77 GDPR)

Right of withdrawal: You can withdraw your consent to Steam login at any time by deleting the website data (localStorage) in your browser. Withdrawal does not affect the lawfulness of processing carried out before the withdrawal.

Right to object: Insofar as processing is based on Art. 6(1)(f) GDPR (legitimate interest), you have the right to object to the processing at any time on grounds relating to your particular situation. However, if you object to the processing of technical connection data, the service can no longer be provided.

Deletion of contributions: Since no separate user account is created, deletion of your data occurs on a contribution-by-contribution basis. You can request the deletion of individual Seeds/Maps submitted by you at any time. To do so, contact me at the address provided above, specifying the affected Seeds/Maps or your Steam ID.

Automated decision-making: No automated decision-making including profiling within the meaning of Art. 22 GDPR takes place.

If you have any questions or concerns about how your data is handled, you can contact me at any time.
